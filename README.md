# dafoam_mcp_server

[![Tests](https://github.com/DAFoam/dafoam_mcp_server/actions/workflows/tests.yml/badge.svg)](https://github.com/DAFoam/dafoam_mcp_server/actions/workflows/tests.yml)
[![codecov](https://codecov.io/gh/DAFoam/dafoam_mcp_server/branch/main/graph/badge.svg)](https://codecov.io/gh/DAFoam/dafoam_mcp_server)

DAFoam MCP server enables conversational pre-processing, simulations, optimization, and post-processing. Currently, we support only airfoil cases with the Claude LLM. The DAFoam MCP server is in the beta state.

Refer to https://dafoam.github.io/ai-agent-overview.html for detailed documentation.
